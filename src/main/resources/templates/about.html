This application can be used to help a fuel attendant quickly calculate the total number of litres they have sold using a pump with at most 6 nozzles. Listed below are other things that this web application can help an attendant do:

  • determine how much total cash sales they have done
  • get a summary of total litres sold
  • quickly calculate how much cash in hand they need to submit which was not dropped

Usage of this application is for learning purposes only. The developer is not responsible for any damages caused. Make sure your input for starting and end readings is correct!
